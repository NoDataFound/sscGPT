`TLDR (Quickstart)`

1. Clone the repo 
```
git clone https://github.com/securityscorecard/ssc-asi-tools.git
cd tools/sscGPT/
```

2. Install requirements
```
python3 -m pip install -r requirements.txt
```

3. Run application
```
./Launch_sscGPT
```

4. Set API keys
